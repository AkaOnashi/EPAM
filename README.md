# Nova Poshta Application
This application allows users to interact with the "Nova Poshta" service for convenient management of shipments, tracking parcels, and accessing current delivery service information.
![Логотип]("D:\gitreps\epam\EPAM\logo.jpg")
## Features
Parcel Tracking: Users can enter the parcel number and obtain information about the current location and status of the parcel.
Shipment Management: Users can create new shipments by providing necessary details about the cargo and recipient's address.
Tariff View: The application provides access to up-to-date tariffs for "Nova Poshta" delivery services.
Branch Locator: Users can view a map with the locations of the nearest "Nova Poshta" branches and obtain additional information about them.
## Requirements
To use the "Nova Poshta" application, you need:

A device running iOS or Android operating system.
Internet connectivity to fetch the latest data from the "Nova Poshta" service.
## Installation
1. Clone the repository to your local machine:
	git clone https://github.com/novaposhta/mobile-app.git
2. Open the project in the appropriate editor.
3. Install dependencies using the package manager for your platform. For example, if you're using npm:
	npm install
4. Run the application on an emulator or connect a physical device for testing.
